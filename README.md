
<h1>Active Directory HomeLab</h1>

<h2>Description</h2>
Project consists of simple Active Directory using Windows Server 2022 and Windows 11 eval versions running in Oracle VM VirtualBox and a PowerShell script to add ~1000 accounts. The purpose of this HomeLab is to gain hands on experience with Windows Server and Active Directory as well as create an environment for future use to develop cybersecurity skills.
<br />


<h2>Utilities Used</h2>

- <b>Oracle VM VirtualBox</b> <a href="Oracle VM VirtualBox">https://www.virtualbox.org/wiki/Downloads</a>
- <b>PowerShell</b> 
- <b>Server Manager</b>

<h2>Environments Used </h2>

- <b>Windows 11</b> <a href="Windows 11 Eval">https://www.microsoft.com/en-us/evalcenter/evaluate-windows-11-enterprise</a>
- <b>Windows Server 2022</b> <a href="Windows Server Eval">https://www.microsoft.com/en-us/evalcenter/evaluate-windows-server-2022</a>
